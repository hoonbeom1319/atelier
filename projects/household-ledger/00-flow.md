## § 화면 목록 (6 — 외부 의존성 없는 독립 HTML, 진입점 index.html)
| 파일 | 화면 | 하단탭? | 역할 |
|---|---|---|---|
| `index.html` | 홈 대시보드 | 홈(활성) | 이번 달 요약·최근 거래·빠른 입력 진입 |
| `add.html` | 거래 입력 | (푸시) | 거래 1건 작성·저장 |
| `list.html` | 거래 내역 | 내역 | 필터·날짜그룹·항목→상세 |
| `detail.html` | 거래 상세 | (푸시) | 단건 보기·수정·삭제 |
| `stats.html` | 월간 통계 | 통계 | 월 이동·도넛·멤버막대 |
| `recurring.html` | 반복 거래 | 반복 | 규칙 목록·추가폼·토글·삭제 |

하단 탭(홈/내역/통계/반복)은 index·list·stats·recurring 4화면에 공통. add·detail은 푸시 화면(상단 ← 뒤로).

## § 사용자 플로우 (A→B·분기)
- **입력:** index → [+ 빠른입력 FAB] → add → [저장] → index (요약 갱신)
- **돌아보기:** index/탭 → list → (필터 칩) → [항목] → detail → [수정]→add / [삭제]→확인→list
- **통찰:** 탭 → stats → [◀▶ 월 이동] / [수입·지출 탭] / [카테고리·멤버 드릴]
- **고정비:** 탭 → recurring → [+ 추가]→인라인 폼→[저장] / [활성 토글] / [삭제]
- **탭 이동:** 4개 탭 화면끼리 하단탭으로 상호 이동.

## § 화면별 구성요소·상태 (← 2층 기능 spec의 출처)

### index.html (홈)
- 구성: 멤버 아바타 줄 / 이번 달 카드(총지출·총수입·잔액) / 최근 거래 리스트(3~5건, 멤버색·카테고리·금액·🔁배지) / FAB(+) / 하단탭.
- 책임 동작: FAB→add 이동(nav). 최근 거래 항목→detail 이동(nav). 하단탭→각 화면(nav).
- 상태: **정상**(거래 있음) / **빈**(거래 0 → "첫 거래를 기록해보세요" + add 유도). 데모 토글로 정상↔빈 전환(dom 변화).

### add.html (거래 입력)
- 구성: ← 뒤로 / 수입·지출 세그먼트(2분기, 단일선택) / 금액 디스플레이+숫자패드 / 카테고리 아이콘 그리드(단일선택) / 날짜(기본 오늘) / 멤버 칩(단일선택, 기본 '나') / 공동·개인 토글 / 메모(선택) / 반복 토글(off→on 시 주기·시작일·종료 노출) / 최근·즐겨찾기 빠른칩 / 저장 버튼.
- 책임 동작:
  - 세그먼트: 지출↔수입 전환 시 `aria-pressed` + 카테고리 세트 교체(dom).
  - 숫자패드: 누르면 금액 디스플레이 증가(dom).
  - 카테고리 타일: 선택 시 `aria-pressed=true`(단일선택, 나머지 false).
  - 멤버 칩: 단일선택 `aria-pressed`.
  - 공동/개인 토글: `aria-pressed` 뒤집힘.
  - 반복 토글: on→반복옵션 영역 표시(dom), off→숨김.
  - 빠른칩(최근/즐겨찾기): 누르면 금액·카테고리 프리필(dom).
  - 저장: 유효(금액>0 & 카테고리 선택)면 활성→누르면 index로(nav). 무효면 `disabled`.
- 상태: **작성중** / **유효성 미충족**(저장 disabled) / **유효**(저장 활성).

### list.html (거래 내역)
- 구성: 헤더(월/전체) / 필터 칩 줄(기간: 이번달·지난달·전체 = 단일 / 카테고리 = 다중토글 / 멤버 = 다중토글 / 공동·개인 = 다중토글) / 날짜 그룹 헤더 / 거래 항목(카테고리 아이콘·메모·멤버색·금액·🔁) / 빈 상태.
- 책임 동작: 필터 칩 토글(`aria-pressed`, 목록 dom 변화) / 항목→detail(nav) / 하단탭(nav).
- 상태: **정상** / **필터 결과 0**(빈 안내, 필터 초기화 버튼) / (로딩은 hi-fi 몫).

### detail.html (거래 상세)
- 구성: ← 뒤로 / 금액 큰 표시 / 유형·카테고리·날짜·멤버·공동개인·메모·출처(수동/🔁자동) / 수정 버튼 / 삭제 버튼.
- 책임 동작: 수정→add(nav) / 삭제→확인 시트 노출(dom)→확인→list(nav) / 뒤로→list(nav).
- 상태: **기본** / **삭제 확인**(시트 열림 dom).

### stats.html (월간 통계)
- 구성: 하단탭 / 월 페이저(◀ 2026.06 ▶) / 요약(수입·지출·잔액 + 전월 대비) / 수입·지출 탭(단일) / 카테고리 도넛(상위5+기타) + 범례 리스트(금액·%) / 멤버별 가로막대(누가 얼마) / 빈 상태.
- 책임 동작: 월 ◀▶(요약·차트 숫자 dom 갱신) / 수입·지출 탭(`aria-pressed`, 차트 교체 dom) / 카테고리 범례 항목 탭(해당 슬라이스 강조 dom, 선택) / 하단탭(nav).
- 상태: **정상** / **데이터 없는 달**(빈 차트 안내). 월 이동으로 정상↔빈 도달 가능.

### recurring.html (반복 거래)
- 구성: 하단탭 / 헤더 / "+ 추가" 버튼 / 규칙 카드 리스트(카테고리·금액·주기·다음 발생일·멤버색 / 활성 토글 / 삭제) / 인라인 추가 폼(닫힘 기본) / 빈 상태.
- 책임 동작:
  - "+ 추가"→인라인 폼 열림(dom), 다시 누르면/취소 닫힘.
  - 폼 저장→리스트에 카드 추가(dom).
  - 활성 토글: `aria-pressed` 뒤집힘(일시중지↔활성).
  - 삭제: 카드 제거(dom).
  - 하단탭(nav).
- 상태: **정상** / **빈**(규칙 0 → 추가 유도).

## § 인터랙션 모델 (구조 결정 — 모양 아님)
- **단일선택:** 수입/지출 세그먼트 · 카테고리 타일 · 멤버 칩 · 통계 수입/지출 탭 · 기간 필터.
- **다중토글:** list 카테고리·멤버·공동개인 필터.
- **on/off 토글(aria-pressed):** 공동/개인 · 반복 on/off · 반복 규칙 활성 · (데모) 정상/빈.
- **인라인 폼 토글:** recurring 추가 폼, detail 삭제 확인 시트.
- **값 페이저:** stats 월 이동(숫자 갱신).
- **숫자패드 누적:** add 금액.
- **프리필:** add 빠른칩.
- 다중선택 일괄액션·드래그·인라인편집은 **범위 밖**.

## § 범위 표시
- **범위 안(MVP):** 위 6화면의 책임 동작 전부.
- **범위 밖(라벨로 명시):** 멤버 초대·권한 관리 / 정산·더치페이 / 예산·한도 / 카드·계좌 연동 / 영수증 OCR / 데이터 내보내기 / 다크모드 토글. 이런 진입점이 화면에 보이면 `[범위 밖]` 라벨(또는 `data-scope="out"`)로 표시 — 죽은 컨트롤 금지.
