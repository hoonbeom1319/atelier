요기조기 — 디자인 인계 패키지 다크모드 · 지도 히어로 피드

장소 중심 사진 SNS(모바일 390×844 · 라이트 · 한국어). 다크모드(검정 배경) UI + 사진은 컬러 실사. chosen final = screens/ 14화면. **홈=피드 하나**(토글 없음), 각 카드가 지도(동선)가 메인 + 작은 컬러 사진. map-home=장소 상세. 업로드는 미분류를 분류 화면에서 카카오맵 검색으로 in-place 배정. 구현 브리프 hand-off.md, 토큰 token-mapping.md, 컴포넌트 component-inventory.md.

사용자 흐름

진입: 로그인 → 온보딩(가치·권한) → 팔로우 시드 → 홈(피드) 홈: 피드 카드(지도 동선 히어로 + 작은 컬러 사진) → 카드 탭 post-detail / POI칩 장소 상세(map-home) 업로드: ＋ → 갤러리 → 자동분류 + 미분류 in-place 배정(카카오맵 검색 / 없는 장소만 직접 핀) → 태그·메모·공개범위 → 게시 → 홈 발견: 검색(카카오 장소/사람/태그) → 장소 상세 · 친구 프로필
