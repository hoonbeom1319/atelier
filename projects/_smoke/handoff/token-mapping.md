# 토큰 매핑표 — _smoke → 대상 시스템

semantic 토큰을 대상 토큰 시스템에 매핑한다(통째 복붙 금지). primitive(`--gray-*`·`--coral-500`)는 값이라 매핑 불필요.

| 우리 semantic 토큰 | 역할 | 대상 시스템 토큰(예시) |
|---|---|---|
| `--color-text` | 본문 텍스트 | `text.primary` |
| `--color-text-muted` | 보조 텍스트 | `text.secondary` |
| `--color-bg` | 페이지 배경 | `bg.canvas` |
| `--color-surface` | 카드 표면 | `bg.surface` |
| `--color-primary` | 브랜드/강조 | `accent.base` |
| `--color-on-primary` | primary 위 텍스트 | `accent.on` |
| `--white` | 흰색 표면값 | `common.white` |
