# collab-kanban — 컴포넌트 인벤토리

> chosen final: `screens/` (index.html + **board.html**). 모든 컴포넌트는 board.html 안에서 조합돼 동작한다.
> 아래 `components/*.html`은 각 컴포넌트의 **상태/변형 쇼케이스**(정적). 실제 상호작용·인라인 JS는 `screens/board.html` 참조.
> 컴포넌트는 semantic 토큰만 사용한다(`token-mapping.md`). 라벨 색만 제품 데이터(`--label-*`).

## 카드 — `card.html`
보드 표면의 컴팩트 카드(정보밀도 절제).
- **표면 노출(이것만):** 제목(2줄 말줄임) · 담당자 아바타 · 라벨칩 1~2 · 마감일 · 댓글수(total). 그 이상은 상세 패널.
- **variant/state:** 기본 / hover(lift 신호) / `dragging`(들림·반투명) / 담당자 있음 / **미배정**(placeholder) / **overdue**(danger 강조) / 긴 제목 2줄 말줄임 / 다중 라벨.
- **쓰이는 화면:** `board.html`(컬럼 카드 리스트), 상세 패널 진입점.

## 컬럼 — `column.html`
보드의 세로 컬럼(status 그룹).
- **구성:** 헤더(이름 + 카드 수 카운트) · 카드 리스트(세로 스크롤) · 하단 "+ 카드 추가".
- **variant/state:** 카드 있음 / **빈 컬럼 드롭존**("여기에 카드를 놓기") / `drop-target` 하이라이트 / 과밀(세로 스크롤) / 추가 입력 펼침.
- **쓰이는 화면:** `board.html`(컬럼 5개: 백로그·할 일·진행 중·검토·완료).

## 필터바 / 아바타 스택 — `filter-bar.html`
보드 상단 필터 + 멤버 가시성.
- **구성:** 담당자 아바타 토글(멀티, +N 오버플로) · "내 카드만" · 라벨 토글 · 마감 세그먼트(오늘/이번주/지남, 단일선택) · "필터 해제".
- **variant/state:** 토글 선택/비선택(`aria-pressed`) · 마감 단일선택(`aria-pressed`) · 활성 필터 시 "필터 해제" 노출 · 아바타 스택 **+N 오버플로**(다중 담당자 확장 대비).
- **쓰이는 화면:** `board.html`(헤더 아래 필터바).

## 카드 상세 패널 — `card-detail.html`
보드 위 오버레이 다이얼로그(`role=dialog`, `aria-modal`).
- **구성:** 제목·설명·담당자(변경)·라벨·마감 + **통합 타임라인**(댓글 + 활동) + 타임라인 필터(전체/댓글/활동) + @멘션 댓글 입력.
- **variant/state:** 타임라인 전체/댓글/활동 · 댓글 항목 vs 활동 항목(톤·아이콘 구분) · @멘션 칩 · 담당자 변경 옵션 펼침 · **뷰어=댓글 입력 숨김**("읽기 전용 — 댓글을 남길 수 없습니다").
- **쓰이는 화면:** `board.html`(카드 클릭 시 오버레이).

## 앱 셸 (두 화면 공유 — 서비스 chrome)
> 사이드바 + 탑바로 보드·홈을 감싸는 전역 프레임. 그라데이션 브랜드(`--gradient-brand`) 셸.
- **사이드바:** 브랜드 마크 + 네비(🏠 홈 / 📋 보드 / ▸ 제품 / 👥 멤버 / ⚙ 설정). **현재 화면 항목 = `aria-current="page"` 비링크**(죽은 링크 방지), 다른 항목 = `a[href]`. 멤버·설정은 `[범위 밖]` 스텁(`data-scope="out"`).
- **탑바:** 검색 input(`검색`, 실동작 [범위 밖]) · `+ 새로`([범위 밖]) · 🔔 알림(드롭다운, `aria-expanded`) · 테마 전환(`aria-pressed`) · 사용자 메뉴 아바타(드롭다운, `aria-expanded`). 보드 화면엔 역할 전환도. 드롭다운 항목 프로필·로그아웃은 `[범위 밖]`.
- **쓰이는 화면:** `index.html`(홈), `board.html`(보드) 둘 다.

## 홈 대시보드 (`index.html` 콘텐츠)
- **구성:** 그라데이션 히어로 인사("안녕하세요, 김지원님") + 위젯 **내게 배정된 카드**(현재 사용자 카드 → 보드 링크) · **최근 활동**(시스템 활동 피드, 정적) · **내 보드**(제품 타일 → board.html / 마케팅 `[범위 밖]`).
- **variant/state:** 라이트/다크 · 위젯 카드 hover lift(`--shadow-bold`) · 보드 타일 vs `[범위 밖]` 타일(점선).
- **쓰이는 화면:** `index.html`(진입).

## 전역 토글(board.html 인라인)
- **역할 토글** 편집자/뷰어 — 뷰어는 순수 읽기전용(드래그·메뉴·추가·편집·배정·댓글 전부 차단/숨김 + "읽기 전용 — 보기만 가능합니다").
- **테마 토글** 라이트/다크 — semantic 토큰 분기, 선택 영속(localStorage `ck-theme`). 양 모드 WCAG AA.

## 접근성 계약(전 컴포넌트 공통)
- 모든 인터랙티브 요소 와이어드(죽은 컨트롤 0). 포터블 셀렉터: role·aria-label·exact text(검증 spec이 이걸로 잡음 — 구현 시 접근 이름 유지).
- 키보드 카드 이동(카드 ⋮ 메뉴 = 키보드 경로) + 네이티브 드래그 시각 상태 병행.
- 라이트/다크 양 모드 텍스트·칩 대비 AA(axe serious 0).
